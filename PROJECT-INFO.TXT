Description:
Glossy is an efficient flooding architecture for wireless sensor networks
that was published at ACM/IEEE IPSN '11.
The source code includes a simple application that periodically floods a
packet and prints related statistics. 
Documentation is available at http://contikiprojects.svn.sourceforge.net/viewvc/contikiprojects/ethz.ch/glossy/doc/html/index.html
